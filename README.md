# Spotify Floating Player (SFL) 🎵

Spotify Floating Player (SFL) adalah ekstensi Chrome premium yang menghadirkan pengalaman lirik mengambang dan pemutar mini langsung di atas tab browser Anda. Dengan desain glassmorphism yang cantik, Anda bisa tetap bernyanyi bersama sambil bekerja di tab lain.

## ✨ Fitur Utama

- **Floating Lyrics Window**: Jendela mengambang yang menampilkan lirik secara real-time.
- **Auto-Scroll Centering**: Lirik aktif secara otomatis terpusat di tengah layar untuk kemudahan membaca.
- **In-App Library Browser**: Jelajahi dan ganti playlist/artis direktori langsung dari dalam player tanpa meninggalkan tab Anda.
- **Global PiP Support**: Mendukung Document Picture-in-Picture untuk lirik yang selalu berada di posisi paling atas (Always on Top) di sistem operasi Anda.
- **Playback Controls**: Kontrol penuh untuk Play/Pause, Next, dan Previous.
- **Glassmorphism Design**: Antarmuka modern yang transparan dan elegan.

## 🚀 Cara Install (Manual)

Karena ekstensi ini masih dalam tahap pengembangan, Anda dapat memasangnya secara manual dengan mengikuti langkah-langkah berikut:

1. **Download & Ekstrak**: Download source code project ini dan ekstrak folder tersebut di komputer Anda.
2. **Buka Extension Page**: Buka browser Chrome dan navigasikan ke alamat `chrome://extensions/`.
3. **Aktifkan Developer Mode**: Nyalakan saklar **Developer mode** yang berada di pojok kanan atas layar.

   ![Developer Mode](<Screenshot 2026-04-22 at 11.46.26.png>)

4. **Load Unpacked**: Klik tombol **Load unpacked** di pojok kiri atas.

   ![Load Unpacked](<Screenshot 2026-04-22 at 11.46.21.png>)

5. **Pilih Folder**: Pilih folder utama hasil ekstrak tadi (folder yang berisi file `manifest.json`).
6. **Selesai!**: Ekstensi SFL Player sekarang sudah aktif dan siap digunakan.

## 📖 Cara Penggunaan

1. Buka [Spotify Web Player](https://open.spotify.com/).
2. Aktifkan mode lirik (ikon mikrofon 🎤) pada Spotify untuk memulai rekaman lirik.
3. Klik ikon ekstensi SFL Player di toolbar browser Anda.
4. Gunakan tombol **Hamburger Menu** (☰) untuk melihat daftar playlist Anda.
5. Gunakan tombol **Square Icon** (❐) untuk mengaktifkan mode Global Floating (Always on Top).

## 🛠️ Teknologi

- JavaScript (Vanilla)
- CSS3 (Glassmorphism & Flexbox)
- Chrome Extension API (Manifest V3)
- Document Picture-in-Picture API
